Browse Site
===========================

> A simple apllication to browse tabs, lists and pages. 

Installing Dependencies
-----------------------

``` bash
npm install
```

Running Locally
---------------

``` bash
# serve with hot reload at localhost:3000
npm run start
```

Test
-----------------------

```bash
# run unit test
npm run test
# run jest
npm run jest
```

[Live demo](https://batuash.github.io/browse-site/)
